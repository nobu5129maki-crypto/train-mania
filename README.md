# 電車マニアクイズ 🚄

電車マニアが挑戦したくなる本格鉄道クイズアプリです。

## 特徴

- **54問**の本格鉄道クイズ（車両形式・路線・鉄道史・技術・用語・ダイヤ）
- **3段階の難易度**（初級 / 上級 / マニア級）
- **6カテゴリ** + 全ジャンルミックス
- **30秒タイマー** + 残り時間ボーナス
- **連続正解ストリーク**でスコアアップ
- **ランクシステム**（見習い駅員 → 伝説の車両基地長）
- 解説付きで鉄道知識が深まる

## 起動方法

```bash
npm install
npm run dev
```

ブラウザで http://localhost:5173 を開いてください。

## ビルド

```bash
npm run build
npm run preview
```

## 遊び方

1. 難易度を選択（初級・上級・マニア級）
2. カテゴリをタップしてクイズ開始（全ジャンルも可）
3. 10問に挑戦！正解すると解説が表示されます
4. スコアを稼いでランクアップを目指そう
